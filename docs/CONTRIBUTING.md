## ✅ 提交类型（Commit Type）

| 类型       | 含义说明                                      |
|------------|-----------------------------------------------|
| `feat`     | 引入新功能                                     |
| `fix`      | 修复 bug                                      |
| `docs`     | 仅文档变更（不包含代码逻辑）                   |
| `style`    | 格式修改（空格、缩进、分号等，不影响逻辑）     |
| `refactor` | 重构（功能不变，优化结构或可读性）             |
| `perf`     | 性能优化                                       |
| `test`     | 添加或更新测试用例                             |
| `chore`    | 其他杂项（构建流程、依赖更新等）               |
| `build`    | 构建相关变更（构建工具、脚本等）               |
| `ci`       | CI/CD 相关变更                                 |
